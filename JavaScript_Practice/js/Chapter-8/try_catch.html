Error Handling:-

No matter how great we are at programming, sometimes our scripts have errors. They may occur because of our mistakes, an unexpected user input, an erroneous server response, and for a thousand other reasons.

For more information About try catch click here

When we apply try catch we will follow the following rules to apply try and catch:-

  1. the code must be runnable. In other words, it should be valid JavaScript.It won't work if the code is syntactically wrong.
  2. If an exception happens in “scheduled” code, like in setTimeout, then try...catch won't catch it.
Throw own errors:-
The "throw" operator generates an error.We can generate own error with the help of throw operator and Error object.

Syntax:-

let error = throw new Error('message')
//or
let err = throw new SyntaxError('message')
let err = throw new ReferenceError('message')
